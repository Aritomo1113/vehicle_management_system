# 車両運行管理システム

訪問看護ステーション向けの車両運行管理システムです。
Streamlitを使用して構築されており、スマートフォンでの操作を想定しています。

## 機能
- **運行記録入力**: 出発・帰社時のメーター、アルコールチェック、給油記録などを入力。
- **管理者ダッシュボード**: 運行記録の一覧表示、CSVエクスポート。
- **車両管理**: 車両の追加・編集、車検日・オイル交換管理。
- **アラート機能**: 車検期限やオイル交換時期を自動で警告。

## 必要要件
- Python 3.12+

## セットアップ & 起動方法

1. 依存ライブラリのインストール
```bash
pip install -r requirements.txt
```

2. 初期データの投入 (初回のみ)
```bash
python seed.py
```
※ `app.db` が作成され、テスト用データが投入されます。

3. アプリケーションの起動
```bash
streamlit run main.py
```
